コーディングとは?

コーディングはプログラミングではない。

htmlやcssはサイトの骨格やデザインを作成するものであって、ロジックを実装するものではないからだ。

webサイトに動きをつけるために用いられるjavascriptはプログラミング言語である。

HTML

webサイトの骨組みを記述する言語。

あくまでも骨格なので、html内ファイル内にstyleタグなどでデザインを記述することは好ましくない。

CSS

webサイトのデザインを記述する言語。

cssは破綻しやすい言語である。webサイトが大規模になるにつれてコードの記述量が増え、煩雑になってしまう。

その弱点を克服するためscssが存在する。

JavaScript

webサイトに動きをつけたり、フロントエンドだけでなくバックエンドの開発を行える言語。

記述しやすくwebサイトだけでなく、サーバー側の開発やネイティブアプリも開発できる。

変数の型が存在しないなど、少々大規模な開発に不向きなところがる。それを克服する形でTypeScriptが存在する。
